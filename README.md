🧭 CourseCompass — AI-Assisted Academic Path Finder

CourseCompass is a smart, browser-based web application that helps students discover the right academic course based on their subject performance and personal interests. Instead of leaving course selection to guesswork, it converts marks and preferences into clear, ranked, and personalized recommendations.

🔗 Live Demo: [https://github.com/Thejas127/course-compass]


✨ Features


📊 Personalized course recommendations with real-time match scoring
✅ Live input validation with subject-wise performance bars
🧠 A "Strength Profile" summarizing academic standing at a glance
💬 Adaptive, encouraging feedback based on performance
🎨 Fully responsive, dark-themed UI with smooth micro-interactions
🛠️ Built entirely with vanilla HTML, CSS, and JavaScript — no external frameworks



🧭 How It Works


Enter your marks — Mathematics, Physics, Chemistry, Biology, English, Commerce, Arts, and Computer Science
Select your stream — Science, Commerce, Arts, or Technology
Choose your interests — Technology, Research, Creative Work, People & Society, and more
Get matched — the app scores and ranks the best-fit courses using a weighted algorithm


Matching Algorithm

Final Score = (Subject Performance × 70%) + (Interest Alignment × 30%)

Courses span multiple fields including Engineering, Medicine, Commerce, Law, Psychology, and Design, each returned with a calculated match percentage.
